Important defaults:

- data directory: `~/.engram`
- database: `~/.engram/engram.db`
- Windows data directory: `%USERPROFILE%\.engram`
- OpenCode config: `~/.config/opencode/opencode.json`
- Gemini config: `~/.gemini/settings.json`
- Codex config: `~/.codex/config.toml`
