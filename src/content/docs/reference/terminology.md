sidebar:
  order: 7
---

- session: a bounded period of work
- observation: a durable memory item
- prompt: a captured user request
- scope: project or personal context
- topic key: stable identifier for evolving memory subjects
- sync chunk: compressed export unit used for Git-based sharing
