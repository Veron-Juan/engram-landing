Representative commands:

- `engram serve [port]`
- `engram mcp`
- `engram tui`
- `engram search <query>`
- `engram save <title> <msg>`
- `engram timeline <obs_id>`
- `engram context [project]`
- `engram stats`
- `engram export [file]`
- `engram import <file>`
- `engram sync`
- `engram projects list`
- `engram projects consolidate`
- `engram projects prune`
- `engram version`
