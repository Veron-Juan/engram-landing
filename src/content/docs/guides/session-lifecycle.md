The lifecycle is simple on purpose.

## Start

A session is created for a project and optional working directory.

## During work

Observations and prompts accumulate as discrete records with project and scope context.

## End

The session summary captures the narrative layer:

- goal
- discoveries
- accomplished work
- relevant files

That structure is what makes the next session legible.
