Use Engram as part of your normal session rhythm.

## During active work

- save decisions when tradeoffs are resolved
- save bug fixes with root cause and affected files
- save discoveries when they would otherwise be expensive to rediscover

## Before ending

Always close with a session summary. That summary is the handoff point for the next session.

## At the next start

Recover project context first, then search if you need a narrower answer.
