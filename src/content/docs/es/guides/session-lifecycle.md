El ciclo de vida es simple a propósito.

## Inicio

Se crea una sesión para un proyecto y, opcionalmente, para un directorio de trabajo.

## Durante el trabajo

Las observaciones y prompts se acumulan como registros discretos con contexto de proyecto y scope.

## Fin

El resumen de sesión captura la capa narrativa:

- objetivo
- descubrimientos
- trabajo realizado
- archivos relevantes

Esa estructura es lo que hace legible la siguiente sesión.
