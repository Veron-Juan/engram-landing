El plugin de Claude Code usa hooks y skills nativos.

Puede:

- asegurar que el servidor esté corriendo
- crear sesiones al inicio
- auto-importar chunks sincronizados
- inyectar contexto previo tras compaction
- mantener disponible de forma consistente el memory protocol
