sidebar:

---

Lanza la TUI con:

```bash
engram tui
```

Comportamientos clave:

- movimiento estilo vim con `j` y `k`
- entrar al detalle con `Enter`
- buscar con `/`
- abrir contexto de timeline con `t`

Esta superficie está optimizada para inspección, no solo automatización.
