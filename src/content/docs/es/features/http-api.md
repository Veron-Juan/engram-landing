La API HTTP refleja el mismo modelo expuesto por otras superficies.

Grupos principales de rutas:

- health
- sessions
- observations
- search
- timeline
- prompts
- context
- export e import

Por defecto, el servidor escucha en `127.0.0.1:7437`.
