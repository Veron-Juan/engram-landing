title: Herramientas MCP
description: Los primitivos de memoria que Engram expone por MCP para agentes y clientes de chat.
sidebar:
  order: 3
---

La superficie MCP incluye herramientas para:

- guardar observaciones
- actualizarlas o eliminarlas
- encontrar contexto reciente
- hacer búsqueda full-text
- iniciar y cerrar sesiones
- capturar prompts
- ver estadísticas y timeline

El sistema está diseñado para que el agente decida qué importa y lo recupere después con mínima fricción.
