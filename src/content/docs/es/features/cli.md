La CLI es el camino más corto para inspeccionar y manipular datos directamente.

Comandos representativos:

- `engram search`
- `engram save`
- `engram context`
- `engram timeline`
- `engram sync`
- `engram projects list`

Usa la sección de referencia para el inventario completo de comandos.
