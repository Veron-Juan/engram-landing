Familias centrales de herramientas:

- observaciones: guardar, actualizar, borrar, obtener
- búsqueda y contexto: search, context, timeline
- control de sesión: start, end, summary
- prompts y estadísticas: prompt save, stats, passive capture, merge projects

Usa esta sección cuando necesites nombres exactos de herramientas y su intención operacional.
