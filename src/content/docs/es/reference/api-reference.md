Endpoints principales:

- `GET /health`
- `POST /sessions`
- `POST /sessions/{id}/end`
- `GET /sessions/recent`
- `POST /observations`
- `GET /observations/recent`
- `GET /observations/{id}`
- `PATCH /observations/{id}`
- `DELETE /observations/{id}`
- `GET /search`
- `GET /timeline`
- `POST /prompts`
- `GET /context`
- `GET /export`
- `POST /import`
