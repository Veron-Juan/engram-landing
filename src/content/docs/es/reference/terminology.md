---
title: Terminología
description: Los términos específicos que usa Engram para sesiones, observaciones, prompts, scopes y artefactos de sync.
sidebar:
  order: 7
---

- sesión: un período acotado de trabajo
- observación: un elemento durable de memoria
- prompt: una solicitud del usuario capturada
- scope: contexto de proyecto o personal
- topic key: identificador estable para temas de memoria en evolución
- sync chunk: unidad de exportación comprimida usada para compartir vía Git
