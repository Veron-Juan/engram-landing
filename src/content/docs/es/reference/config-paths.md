Defaults importantes:

- directorio de datos: `~/.engram`
- base de datos: `~/.engram/engram.db`
- directorio de datos en Windows: `%USERPROFILE%\\.engram`
- config de OpenCode: `~/.config/opencode/opencode.json`
- config de Gemini: `~/.gemini/settings.json`
- config de Codex: `~/.codex/config.toml`
