---
title: Variables de Entorno
description: Las pocas variables que controlan el path de datos, el puerto y la resolución de proyecto en Engram.
sidebar:
  order: 5
---

- `ENGRAM_DATA_DIR`: cambia el directorio de datos
- `ENGRAM_PORT`: cambia el puerto de la API HTTP
- `ENGRAM_PROJECT`: sobrescribe la detección de proyecto en modo MCP
